## 🔑 Get Your FREE Groq API Key

1. Visit: **https://console.groq.com/keys**
2. Click **"Sign in with Google"**
3. Click **"Create API Key"**
4. Give it a name (e.g., "RuleWise")
5. Click **"Submit"**
6. **Copy the key** (starts with `gsk_...`)

---

## 📝 Add Key to Your App

1. Open: `c:\xampp\htdocs\RuleWise\lib\services\ai_service.dart`

2. Find line 7:
```dart
static const String _groqApiKey = 'YOUR_GROQ_API_KEY_HERE';
```

3. Replace with YOUR key:
```dart
static const String _groqApiKey = 'gsk_YOUR_ACTUAL_KEY_HERE';
```

4. Save the file

---

## 🔄 Restart the App

**IMPORTANT:** Stop all running instances first!

1. In BOTH terminals, press `q` to quit
2. Run: `flutter run`
3. Wait for app to load
4. Test AI Assistant!
